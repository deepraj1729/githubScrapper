# githubScrapper
A simple github Scrapper to retrieve data from github to automate tasks using:

    - BeautifulSoup
    - Requests

### To install:
    
    git clone https://github.com/deepraj1729/githubScrapper.git
    
    cd githubScrapper/
    
    pip install -r requirements.txt

### To Run:
Windows users:

    python main.py

Linux Users:

    python3 main.py
